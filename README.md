# modem
------------------------------------------------
x/y/z modem 协议，主要目标：

* 抽象出抽传输层，方便移植，方便调试和测试。
* 支持进度上报，方便提供GUI的界面。 

install:
------------------------------------------------

```
git clone https://github.com/xianjimli/modem
git submodule update --init --recursive
```

build:
------------------------------------------------

```
make
```
